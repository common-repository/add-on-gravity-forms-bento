== Description ==

This community plugin allows you to connect your forms created in Gravity Forms to the Bento marketing automation platform.

== Installation ==

1. Upload & activate the plug-in

2. Configure plugin settings via Forms > Settings > Bento

3. For each form you wish to connect, visit Form > Settings > Bento to configure feeds.
